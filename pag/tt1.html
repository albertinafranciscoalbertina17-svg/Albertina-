CONTEÚDO: INTRODUÇÃO AO CSS

1. O que é CSS?

CSS significa Cascading Style Sheets (Folhas de Estilo em Cascata). É uma linguagem usada para dar estilo às páginas Web criadas com HTML. Enquanto o HTML serve para estruturar o conteúdo (textos, imagens, botões), o CSS é responsável pela aparência visual da página.

Exemplos do que o CSS faz:

• Mudar cores dos textos;
• Alterar o tamanho das letras;
• Definir cores de fundo;
• Posicionar elementos na página.

2. Importância do CSS no Desenvolvimento Web

O CSS é muito importante porque:
• Melhora a aparência dos sites, tornando-os mais bonitos e organizados;
• Facilita a leitura do conteúdo;
• Permite a separação entre conteúdo e design:
  HTML → conteúdo
  CSS → aparência;
• Facilita a manutenção e actualização dos sites;
• Garante uma melhor experiência do utilizador.

3. Tipos de Folhas de Estilo CSS

Existem três formas principais de aplicar CSS em uma página HTML.

3.1 CSS Inline

O CSS inline é aplicado directamente dentro da própria tag HTML.

<p style="color: blue;">Este texto é azul</p>

Características:
• Simples e rápido;
• Usado para pequenas alterações;
• Não recomendado para páginas grandes.

3.2 CSS Interno (Internal CSS)

O CSS interno é escrito dentro da própria página HTML, usando a tag <style>.

<style>
p {
  color: green;
}
</style>

Características:
• Organiza melhor o código que o CSS inline;
• Aplica estilos a vários elementos ao mesmo tempo;
• Usado quando o estilo é apenas para uma página.
